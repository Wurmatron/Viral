## Overview

Viral is a mod that adds a spreading virus called "Viral". Viral spreads from mob to mob while buffing hostile mobs and damaging animals.  This mod only has one feature, the spreading of the virus "viral" from mob to mob. This mod also adds a few items that help with controlling the spread.

## Features

- Spreading "Virus" Weakens Passive, and makes Hostile stronger.
- Ability to cure and make mobs immune.
- Repel mobs (via mob drops)
- Interdiction to prevent mobs from getting near your farms.
